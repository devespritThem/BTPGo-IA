Roadmap (proposée)

Phase 1 — MVP rentable
- Auth + Orgs + JWT + 2FA
- Projets + Gantt + Planification/Forecast heuristique
- Finance de base: factures, dépenses, paiements, cashflow, profitabilité
- OCR AO + extraction, Messaging projet + résumé IA
- PWA mobile: QR scan, offline

Phase 2 — Croissance
- Marketplace (matching/commissions), Financement/Assurance (partenaires)
- Maintenance (tickets/garanties/satisfaction), Subscriptions (Stripe)
- ESG: reporting CO₂, objectifs
- Monitoring: Dashboards Grafana officiels, alerting Prometheus
- Sécurité: RBAC fin, scans SAST/DAST, Trivy bloquant

Phase 3 — Avancé
- IA générative (LLM) pour devis/estimation, NL Q&A data warehouse
- Intégrations ERP/Compta (Sage, Cegid), SSO entreprise
- Multi‑régions, HA, autoscaling, feature flags, audit avancé
