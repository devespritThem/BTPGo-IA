Durabilité & ESG (Carbone)

Concepts
- EmissionFactor: facteur kgCO2e par unité (matériaux, énergie, transport)
- ProjectEmission: enregistrements (projet, facteur, quantité)

Endpoints
- `POST/GET /esg/factors`
- `POST /esg/projects/emissions`, `GET /esg/projects/:projectId/emissions`
- `GET /esg/projects/:projectId/recommendations`

Extensions
- Facteurs par pays/fournisseur, import Base Carbone ADEME
- Corrélation avec coûts/planification, objectifs de réduction
