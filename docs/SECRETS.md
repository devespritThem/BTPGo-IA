Secrets Reference

Backend
- JWT_SECRET: JWT signing secret
- DATABASE_URL: Postgres URL (prod)

Fly.io
- FLY_API_TOKEN: Personal access token for deploy

o2switch
- O2SWITCH_FTP_HOST: FTP hostname
- O2SWITCH_FTP_USER: FTP username
- O2SWITCH_FTP_PASS: FTP password
- O2SWITCH_FTP_PATH: Remote path (e.g., /public_html)

AI Providers (optional)
- OPENAI_API_KEY, etc. (when integrated in `ai_engine`)
